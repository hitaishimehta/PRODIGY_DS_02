# 📊 PRODIGY_DS_02 - Exploratory Data Analysis

## 📌 Task Objective

Perform data cleaning and exploratory data analysis (EDA) on a dataset to identify patterns, relationships, and trends.

---

## 📂 Dataset

This project uses a movie dataset containing:

* Title
* Star Rating
* Genre
* Duration
* Content Rating
* Actors

---

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔍 Steps Performed

1. Loaded dataset using Pandas
2. Checked dataset structure (`.info()`, `.shape`)
3. Handled missing values and duplicates
4. Performed statistical analysis
5. Visualized data using graphs

---

## 📈 Key Insights

* Most movies fall within a similar rating range
* No strong correlation between duration and ratings
* Some genres appear more frequently than others
